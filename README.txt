Резервное копирование
Чтобы настроить работу программы перейдите в файл Program\Settings\settings.json
Для добавления исходных папок добавьте значение в ключ fromDir (учтите, что можно добавлять несколько папок - "fromDir": ['C:\\File 1', 'C:\\File 3'])
Для добавления целевой папки воспользуйтесь ключом toDir - например "toDir": 'C:\\File 2'
!!!ВНИМАНИЕ!!! Слэши обязательно должны дублироваться - \\.
После настройки файла конфигурации запустите приложение TestApp1
